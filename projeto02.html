Análise sobre locações da rede Airbnb Amsterdã

Nesse projeto foi realizada uma análise de dados exploratória buscando indicar possíveis oportunidades e/ou ameaças para o empreendimento.

Estudo de caso

- A análise foi realizada com base nos dados do Airbnb de Amsterdã, contendo informações sobre os preços, como o número de quartos, tipo de quarto, status de host, distância do centro da cidade e outros atributos.

- Importante ressaltar que a base de dados analisada é a base semanal para dias úteis (terça à quinta-feira).
